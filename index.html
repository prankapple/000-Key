000 Key

000 Key is a Kill button using the 000 button on a numpad keyboard extension.

To set up, download the main application and place it in the STARTUP folder, download the Kill application and put it in a easy to find location (eg. Desktop)

When the computer restarts and you click the 000 key all non-critical processes will be terminated. The kill app terminates the main app.
